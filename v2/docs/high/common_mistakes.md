# 高中常见错误与纠错话术（high）

## 函数
- 错误：不写定义域就直接讨论单调/最值
  纠错：先确定 x 的取值范围，否则结论可能不成立。
- 错误：分类讨论漏情况（例如 x>0, x<0, x=0）
  纠错：把边界点单独检查，确保不遗漏。
- 错误：画图凭感觉，不核对关键点（截距、顶点）
  纠错：至少算出关键点再画图。

## 二次函数/方程
- 错误：判别式 Δ 计算错误
  纠错：Δ=b^2-4ac，注意括号和符号。
- 错误：求根后不验证原式条件（分母不为 0、根号内非负）
  纠错：把根代回“限制条件”检查。

## 三角
- 错误：恒等变形方向乱，导致越变越复杂
  纠错：先明确目标（化成同一函数/同一角），再选择恒等式。
- 错误：角度单位混乱（度/弧度）
  纠错：确认题目单位，保持一致。

## 数列
- 错误：把等差当等比（或相反）
  纠错：先看相邻差是否固定，再看相邻比是否固定。
- 错误：求和公式使用条件不满足
  纠错：先确认确实是等差/等比，再套公式。

## 概率
- 错误：条件概率当成普通概率
  纠错：题目说“在…条件下”，就要用 P(A|B) 的思路（资料中需有公式）。
