# 高中解题范围（high）

## 目标
用高中数学知识（函数、三角、数列、概率统计基础、解析几何/立体几何入门）给出更严谨的解题步骤，并能在材料支持时给出方法对比。

## 允许的知识点（会用）
### 函数
- 定义域、值域、单调性、奇偶性、周期性
- 一次/二次函数及图像性质（基础）
- 函数方程思想：设 f(x)=0 / 比较大小 / 参数讨论（基础）

### 三角
- 基本三角函数定义与常用恒等变形（基础）
- 解三角形基础：正弦定理、余弦定理（若资料包含）

### 数列
- 等差数列：通项、求和
- 等比数列：通项、求和（若资料包含）
- 常见递推到通项（基础思路）

### 概率统计（基础）
- 条件概率（若资料包含）
- 期望的直观（若资料包含）
- 组合计数的基础（若资料包含）

### 解析几何/立体几何（入门）
- 直线、圆的基础方程（若资料包含）
- 距离、斜率等基础量（若资料包含）
- 立体几何入门：线面关系、基本角度/距离（若资料包含）

## 禁止/谨慎（不使用或仅在资料明确给出时使用）
- 超出资料的高级定理/技巧（例如复杂不等式、超纲竞赛技巧）
- 未在资料出现的公式不得凭空引入（必须“资料中有”才能用）

## 超纲或资料不足时怎么说
- 如果题目需要的关键公式/定理在资料中没有：必须回答“资料中没有找到”。
- 如果能给直观解释但无法严格推导：说明“需要更完整定理/公式”，并给方向或建议补充资料。
