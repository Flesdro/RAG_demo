# 小学术语与同义词（primary）

## 平均
同义词：均分、平分、每份一样多
解释：把东西分成一样多的几份。
常见问法：每人分到多少？平均每份多少？

## 每
同义词：每个、每盒、每袋、每小时
解释：“每”常表示要用乘法或除法。
常见问法：每盒彩笔多少钱？每小时走多少千米？

## 一共 / 总共
同义词：合起来、总计、全部
解释：把几部分加在一起。
常见问法：一共多少钱？总共多少人？

## 还剩 / 剩下
同义词：剩余、还剩下、没用完
解释：从总数里减去用掉的部分。
常见问法：还剩多少？还剩几本？

## 相差
同义词：差多少、多多少、少多少
解释：用减法比较两个数。
常见问法：A 比 B 多多少？差几米？

## 单价
同义词：每个多少钱、每件多少钱
解释：一个东西的价格。
关系：单价 × 数量 = 总价

## 数量
同义词：多少个、几盒、几本
解释：东西的个数。
关系：单价 × 数量 = 总价

## 总价
同义词：一共花了多少钱、总共多少钱
解释：买这些东西花的全部钱。
关系：单价 × 数量 = 总价

## 周长
同义词：一圈的长度、围一圈有多长
解释：图形边边加起来的长度。

## 面积
同义词：占地大小、铺满需要多少
解释：图形内部有多大（用平方单位）。
