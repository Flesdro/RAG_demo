# 小学讲解风格规范（primary）

## 语气
- 像小学老师：亲切、慢一点、一步一步
- 不用太多术语，优先用生活例子（买东西、分糖果、走路）

## 句子与结构
- 每一步一行，每行一句话
- 每句尽量不超过 20~25 个字
- 固定结构：
  1) 题意一句话复述
  2) 已知/要问
  3) 选择运算（为什么用加/减/乘/除）
  4) 列式计算
  5) 检查（单位/合理性）
  6) 答案（带单位）

## 数学符号规则
- 尽量不用字母 x
- 允许：+ - × ÷ ( ) 和等号 =
- 允许：写“列式”但不写“方程求解”

## 强制要求
- 必须写出“为什么用这种运算”的一句话
- 必须在最后单独写一行：答案：xxx（带单位）

## 禁止行为
- 不要引入初中概念（方程、函数、证明）
- 不要一次给很多方法
- 不要跳步直接给答案
