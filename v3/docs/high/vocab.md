# 高中术语与同义表达（high）

## 定义域
同义词：自变量取值范围
解释：能让表达式有意义的 x 的范围。
常见问法：定义域是什么？x 能取哪些值？

## 单调性
同义词：递增/递减趋势
解释：x 变大时函数值是否一直变大（单调增）或变小（单调减）。
常见问法：函数单调吗？在哪个区间增/减？

## 奇偶性
解释：f(-x)=f(x) 为偶函数；f(-x)=-f(x) 为奇函数。
常见问法：判断奇偶性

## 周期
解释：存在 T>0 使 f(x+T)=f(x)，最小正 T 为周期。
常见问法：周期是多少？

## 判别式 Δ
解释：二次方程 ax^2+bx+c=0 的 Δ=b^2-4ac，用于判断实根情况。
常见问法：有几个实根？根的情况？

## 等差数列
解释：相邻两项差相等，公差为 d。
常见问法：通项 a_n？求和 S_n？

## 等比数列
解释：相邻两项比相等，公比为 q。
常见问法：求第 n 项？前 n 项和？

## 正弦定理/余弦定理
解释：解三角形用的基本定理（若资料包含才能用）。
常见问法：已知两边一角求第三边？求角？

## 条件概率
解释：在事件 B 已经发生的条件下事件 A 发生的概率 P(A|B)（若资料包含）。
常见问法：已知…情况下…概率？
