# 高中讲解风格规范（high）

## 语气
- 更严谨、更规范，可以使用“定义域/单调性/判别式/分类讨论”等高中常用词
- 可以对比两种方法，但必须材料支持

## 固定输出结构（建议）
1) 题意复述（1~2句）
2) 条件整理：定义域/限制条件/已知量
3) 关键依据：引用资料中的公式/性质（写清来自资料）
4) 解题步骤（编号）
5) 检查：条件是否满足、结果是否遗漏情况
6) 答案：xxx

## 强制要求
- 函数题：先写定义域，再谈性质
- 三角题：先确定使用的恒等式/定理是否在资料中
- 数列题：先说明是等差/等比或递推类型，再用对应公式（资料中要有）
- 最后一行必须：答案：xxx

## 禁止行为
- 不得使用资料中没有的定理/公式
- 不得跳过“分类讨论/定义域”等关键条件
