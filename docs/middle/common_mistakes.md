# 初中常见错误与纠错话术（middle）

## 代数与方程
- 错误：移项不变号
  纠错：移项等于两边同时加/减同一个量，所以符号会变。
- 错误：去括号时忘记“-”要变号
  纠错：-(a-b)= -a + b，括号内每一项都要变号。
- 错误：合并同类项把不同字母也合并
  纠错：只有字母部分完全相同才是同类项。
- 错误：解出 x 后不检验
  纠错：把 x 代回原方程看等号是否成立。

## 方程组
- 错误：代入后漏掉括号
  纠错：代入的是“整体”，必要时先加括号再展开。
- 错误：加减消元时符号处理错
  纠错：先把两式写成标准形式，再逐项相加/相减。

## 几何证明
- 错误：没写“用哪个判定/性质”
  纠错：证明全等/相似必须点明判定（SAS/ASA/AA 等）。
- 错误：对应关系乱了
  纠错：先写清楚对应点，再写对应边角。

## 一次函数
- 错误：把斜率 k 当成截距 b
  纠错：k 是 x 的系数；b 是 x=0 时 y 的值。
- 错误：过两点求直线时算错 k
  纠错：k=(y2-y1)/(x2-x1)，先减再除，不要颠倒。

## 概率
- 错误：把“有利数”当成“总数”
  纠错：总数是所有等可能结果的数量，有利数是符合条件的那部分。
