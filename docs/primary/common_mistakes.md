# 小学常见错误与纠错话术（primary）

## 审题类
- 错误：漏看“每/平均/还剩/一共”
  纠错：把关键词圈出来，它决定用什么运算。
- 错误：把“还剩”当成“又多了”
  纠错：“还剩”表示要从总数里减掉用掉的。

## 运算类
- 错误：先加后乘导致顺序错
  纠错：一般先算“每份×份数”（乘除），再把结果加起来。
- 错误：计算时忘记进位/借位
  纠错：把竖式写出来，再算一遍。

## 单位类
- 错误：答案不写单位（元、米、个）
  纠错：最后一定要带单位，不然不完整。
- 错误：单位没换就直接算
  纠错：先把单位换成一样的，再计算。

## 合理性检查
- 错误：买 3 盒每盒 12 元，算成 12÷3
  纠错：盒数变多，总价应该变大，所以用乘法更合理。
